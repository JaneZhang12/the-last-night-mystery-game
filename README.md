# 最后一夜（The Last Night）

一款在浏览器中运行的单页互动解谜游戏。

玩家将在雨夜收到一封神秘委托，探索旧宅中的不同场景，收集线索并解开谜题，最终翻开日记本，抵达故事的结局。
日记本的照片很适合放置小惊喜（比如生日、纪念日等等）。

## 游戏特色

- 单文件 HTML 游戏，无需安装或构建
- 场景热点探索与线索收集
- 五道互动谜题
- 自定义侦探代号
- 立体翻页纪念册与信件效果
- 本地背景音乐、字体和场景素材
- 支持重新开始游戏

## 开始游戏

下载项目内的`所有`文件后，使用现代浏览器打开 `The Last Night.html` 即可游玩。

HTML 中的素材引用均使用相对路径，因此无论将项目放在哪个磁盘或目录，通常都不需要修改路径。请完整保留项目的目录结构，不要单独移动 HTML、`fonts/`、`pics/` 或背景音乐文件；如果重命名或移动了任何素材，请同步修改 HTML 中对应的相对路径。

> 浏览器通常需要一次用户点击后才允许播放背景音乐。可使用页面右下角的音乐按钮控制播放。

## 操作方式

1. 输入侦探代号并签名。
2. 点击场景中的热点调查线索。
3. 收集完当前场景的线索后进行推理。
4. 选择谜题答案，推进故事。
5. 完成最后一道谜题，阅读纪念册与结尾信件。

游戏主要使用鼠标操作，也可通过键盘填写侦探代号。

## 项目结构

```text
.
├─ The Last Night.html       # 游戏页面、样式与逻辑
├─ Attic Whisper.wav         # 背景音乐
├─ fonts/                    # 本地字体
├─ pics/
│  ├─ scene/                 # 场景图片
│  └─ tools/                 # 道具图片
└─ README.md
```

## 自定义内容

- 场景与谜题内容位于 `The Last Night.html` 中。
- 场景图片位于 `pics/scene/`。
- 道具图片位于 `pics/tools/`。
- 背景音乐为 `Attic Whisper.wav`。
- 纪念册当前使用三页中性占位文字，可替换为合适的内容。

修改素材文件名或目录后，请同步更新 HTML 中对应的相对路径。

## 兼容性

建议使用较新版本的 Chrome、Edge、Firefox 或 Safari。游戏采用原生 HTML、CSS 和 JavaScript 编写，不依赖第三方框架。
